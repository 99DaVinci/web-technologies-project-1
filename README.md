Project Title
Web Technologies Project 1: HTML and CSS Layouts

Brief Project Description
This project demonstrates the power of Cascading Style Sheets (CSS) by utilizing a single, valid XHTML 1.0 Strict document (index.html) that drastically changes its layout and appearance depending on the linked stylesheet.

The project includes:

Style A: A responsive vertical layout utilizing Flexbox to evenly space elements without overlapping.

Style B: A horizontal layout demonstrating inline-block positioning and fixed elements.

Style C: A modernized layout featuring advanced CSS properties such as smooth hover transitions, interactive scaling, and box shadows.

Setup / Run Instructions
Save all project files (index.html, styleA.css, styleB.css, styleC.css) into a single folder on your computer.

Open the index.html file using the Google Chrome web browser to ensure full compatibility.

To switch between styles: Open index.html in a text editor. In the <head> section, you will see the <link> tags for the stylesheets. Uncomment the one you want to view and comment out the others (using ``), then save and refresh your browser.
